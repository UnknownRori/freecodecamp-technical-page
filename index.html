Introduction to Python

Python is a popular programming language. Python can be used on a server to create web applications. Python is a popular programming language. It was created by Guido van Rossum, and released in 1991. It is used for:

  • Web Development (Server Side)
  • Software Development
  • Mathematics
  • System Scripting
  • Game Development

First python syntax to print out text : print()

For example :

print("Hello, World!")

Python was designed for readability, and has some similarities to the English language with influence from mathematics, Python uses new lines to complete a command, as opposed to other programming languages which often use semicolons or parentheses, Python also relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes. Other programming languages often use curly-brackets for this purpose.

Python Installation

If your computer already installed Python you might want to check it's version by typing python --version in command line.
If your computer doesn't have Python you can use this following procedure to download

  • First open browser and visit python.org
  • On download dropdown you can download python
  • After finished downloading you can run the installer
  • After finished installing open command line and then write python --version and pip --version to check whether or not Python installed correctly

To run your python file you will need to type python yourfilename.pyin command line

Python Comment

There are two type of comment in python first is one line comment and multi-line comment:

# This is comment """ Multi line comment """

Python Variable

Variables are containers for storing data values. Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.

x = 5

Variables do not need to be declared with any particular type, and can even change type after they have been set.

y = "John"

Python Datatype

In programming, data type is an important concept. Variables can store data of different types, and different types can do different things. Python has the following data types built-in by default, in these categories:

Data Type Type
Text Type Str
Numberic Type int, float, complex
Sequence Type list, tuple, range
Mapping Type dict
Set Type set, frozenset
Boolean Type bool
Binary Type bytes, bytearray, memoryview

You can get the data type of any object by using the type() function:

x = 5

print(type(x))
